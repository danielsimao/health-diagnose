## Description

Monorepo CRUD Application.

**Client:** TailwindCSS, Next.js

**Server:** Nest.js

**NOTICE:** Before running the app you may need to set .env files, both client and server. Please follow the .env.examples provided.

## Installation

```bash
$ npm run bootstrap
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Troubleshooting

```bash
# full clean-up
$ npm run clean
```
