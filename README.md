## Description

Monorepo CRUD Application.

Client: TailwindCSS, Next.js

Server: Nest.js

## Installation

```bash
$ npm run bootstrap
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Troubleshooting

```bash
# full clean-up
$ npm run clean
```
